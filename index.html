How we work this out. Every team has separate attack and defence ratings based on their results — better teams score more and concede less. We use these ratings to simulate each match by sampling goals directly from a statistical distribution, rather than deciding a winner first and fitting a scoreline after. We run the whole tournament 100,000 times to build up the probabilities you see here.

What you see in the bracket is one likely path — the most probable outcome at each match, not a guarantee. For the full picture — every team's chance of reaching each stage — see the full table.

Results already played are built in and update each team's ratings as the tournament progresses, so predictions improve as more matches are played.
